git的实现原理：先创建文件夹，然后通过git GUI 到提交代码、解决冲突的中转站，然后到达本地仓库（不能联网的情况下提交的代码先至该出），最后到远程仓库即github网站的仓库。
需将远程仓库与文件同步，要通过git Bash，输入 git clone 加网址，将文件和远程仓库同步，远程仓库的代码更新到本地，通过git fetch更新到本地仓库，后就进入到工作区间。
   远程仓库---git fetch ----本地仓库 -----git reset -----缓存区-----git checkout-----工作区间-----git add files ----缓存区 -----git commit ----本地仓库--git push ----远程仓库
